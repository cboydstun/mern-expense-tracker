## MERN Expense Tracker

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:cboydstun/mern-expense-tracker.git
   ```
2. Install Client Packages
   ```sh
   cd client
   ```
   ```sh
   yarn install
   ```
   ```sh
   yarn start
   ```
3. Install Server Packages

   ```sh
   cd server
   ```

   ```sh
   yarn install
   ```

   ```sh
   yarn start
   ```

   - Remember DOTENV variables in .env.sample and then remove .sample! =)
